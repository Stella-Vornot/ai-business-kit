## 📋 Detailed Setup: AI Sales Agent Blueprint

### Prerequisites (5 minutes)
1. Sign up for Make.com (free tier: 1,000 operations)
2. Sign up for VAPI ($5 free credit)
3. Get OpenAI API key ($5 free credit)

### Step 1: Import Make.com Workflow (10 minutes)
1. Log into Make.com
2. Click "Create new scenario"
3. Click "Import blueprint"
4. Upload `ai_sales_agent_blueprint_WORKFLOW.json`
5. Connect your apps (follow prompts)

### Step 2: Customize VAPI Prompt (15 minutes)
1. Log into VAPI
2. Create new assistant
3. Copy prompt from `VAPI_PROMPT_TEMPLATE.md`
4. Replace placeholders (your company name, service, etc.)
5. Test with 5 calls

### Step 3: Connect Everything (20 minutes)
1. Copy VAPI assistant ID into Make.com
2. Add your Calendly link
3. Add your email for notifications
4. Test complete flow with test lead

### Step 4: Go Live (5 minutes)
1. Add webhook URL to your website form
2. Turn on Make.com scenario
3. Monitor first few calls
4. Adjust VAPI prompt if needed

**Total setup:** 55 minutes  
**First results:** Usually within 24 hours

## 🆘 Troubleshooting

### Common Issues & Solutions

**"Make.com scenario not running"**
- Check if toggled ON
- Verify webhook URL is correct
- Test trigger manually

**"VAPI calls not connecting"**
- Check phone number format (+1 for US)
- Verify VAPI has credit
- Check assistant is active

**"Emails not sending"**
- Verify Mailchimp connection
- Check email isn't going to spam
- Test with different email address

**"Low booking rate"**
- Review VAI call recordings
- Adjust conversation prompts
- Add more qualification questions
- Follow up sooner in email sequence
